# Code of Conduct

This project aims to be welcoming, respectful, and collaborative.

## Our standards

- Be respectful and constructive.
- Assume good intent and focus on technical issues.
- Avoid harassment, discrimination, and personal attacks.

## Unacceptable behavior

- Harassment, hate speech, or abusive language.
- Trolling, personal insults, or repeated disruptive behavior.
- Publishing private information without consent.

## Reporting

If you experience or witness unacceptable behavior, please report it to the project maintainer via GitHub:

- Open an issue if appropriate: https://github.com/simoninns/decode-orc/issues
- For sensitive reports, contact the maintainer directly: https://github.com/simoninns

Reports will be reviewed and handled in good faith.

## Enforcement

The maintainer may edit or remove comments, issues, pull requests, or other contributions that violate this Code of Conduct, and may temporarily or permanently restrict participation where necessary.
